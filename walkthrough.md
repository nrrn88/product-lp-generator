# 商品詳細ページ自動生成ツール (Web UI版) 利用ガイド

## 1. アプリの起動
既に背後で起動していますが、もし停止している場合は以下のコマンドで起動します。
```powershell
cd C:\Users\lotus\.gemini\antigravity\scratch\product_generator
streamlit run app.py
```

## 2. 画面へのアクセス
ブラウザで以下のURLを開いてください。
**[http://localhost:8501](http://localhost:8501)**

## 3. 利用手順

### 手順①: APIキーの設定
1.  画面左側のサイドバーに「Gemini API Key」入力欄があります。
2.  Google AI Studioで作成したAPIキーを入力してEnterを押してください。
    *   キーが無い場合は [Google AI Studio](https://aistudio.google.com/app/apikey) で「Create API key」から無料で作成できます。

### 手順②: 情報の入力
1.  **商品名**: 生成したい商品名を入力（例: アナドリン）。
2.  **参考にする競合URL**:
    *   参考にしたい他社サイトのURLを貼り付けます。
    *   複数ある場合は改行して入力してください。
3.  **特記事項**:
    *   「成分量は50mgにして」「配送は在庫あり即納」など、特に含めたい情報があれば追記します。
4.  「**🚀 ページを生成する**」ボタンをクリックします。

### 手順③: 確認とダウンロード
1.  ボタンを押すと、AIが競合情報を読み込み、数秒〜数十秒でHTMLを生成します。
2.  画面右側のプレビューエリアに結果が表示されます。以下のタブで表示を切り替えられます：
    *   **🖼️ プレビュー**: 生成されたHTMLの見た目を確認
    *   **📝 HTML**: HTMLソースコードの確認とダウンロード
    *   **⚙️ メタデータ**: タイトル、ディスクリプション、画像プロンプト
    *   **⭐ レビュー**: 生成されたユーザーレビュー（JSON形式とプレビュー）
    *   **🔗 参考リンク**: 生成に使用した信頼できる情報源リンク
3.  問題なければ、「**💾 HTMLファイルをダウンロード**」ボタンで保存します。

## 補足: 出力されるHTMLについて
*   **構造化データ入り**: 検索エンジン(AIO)対策として、JSON-LD形式の `Product`、`FAQPage`、`AggregateRating`（レビュー評価）が含まれています。
*   **スタイル**: プレビューでは簡易表示ですが、実際のECサイトに組み込めば、サイト共通のCSSが適用されます。
